What is collection frameworks?
It provides a set of interfaces and classes that help in managing group of object.

Before the introduction of the collection framework in JDK 1.2, Java used to rely on a variety of classes like Vector, Stack,
Hashtable, and arrays to store and manipulate group of objects.
